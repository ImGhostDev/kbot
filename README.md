# KBOT
There is a simple telegram bot.
It can answer on messages like "hello", and "owner".

link: t.me/NaSA_Allergy_bot
# Instructions
### how to build:
>> go build -ldflags "-X="github.com/ImGhostDev/kbot/cmd.appVersion=v1.0.2
### how to run:
>>  ./kbot start 
### version chech:
>>  ./kbot version
